# Чек-лист самопроверки: Спринт 8

- [ ] Приложение компилируется и запускается на эмуляторе.
- [ ] Старт приложения начинается с экрана, на котором размещено меню для перехода на экраны Songs, Playlists, Favorites.
- [ ] При переходе с главного экрана на экран Songs отображается поисковая строка. При наборе символов в поисковой строке начинается поиск треков, соответствующих введённому запросу. В начале поиска отображается лоадер. При завершении поиска лоадер исчезает и на экране отображается список найденных треков или сообщение, если ничего не было найдено.
- [ ] При клике на трек происходит переход на экран детального описания трека. Экран детального описания трека помимо данных о треке имеет кнопки для добавления композиции в «Избранное» и в существующий плейлист.
- [ ] При переходе с главного экрана на экран Playlists отображаются список существующих плейлистов и кнопка для создания нового плейлиста. При нажатии на кнопку создания нового плейлиста осуществляется переход на экран создания нового плейлиста.
- [ ] Экран создания нового плейлиста имеет возможность задать имя и описание этого плейлиста, также здесь размещена кнопка для его сохранения в хранилище.
- [ ] При переходе с главного экрана на экран Favorites отображается список всех треков, добавленных в «Избранное».
