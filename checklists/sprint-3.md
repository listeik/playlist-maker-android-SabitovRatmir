# Чек-лист самопроверки Спринт 3

## Задача 1 Главный экран приложения
- [ ] Приложение запускается и показывает главный экран, свёрстанный по дизайн-макету.
- [ ] При нажатии на кнопки показывается Toast.

## Задача 2 Переходы между экранами
- [ ] В приложении появились экраны настроек и поиска.
- [ ] В обработчиках нажатия на соответствующие кнопки нет Toast.
- [ ] При нажатии на кнопки выполняется открытие экранов.
